TOPIC
    about_FarInventory

SHORT DESCRIPTION
    Computer inventory tools for Far Manager.
    Author: Roman Kuzmin

LONG DESCRIPTION
    This module tools get and show various information about the computer. Some
    types of returned objects are only for viewing. Some others allow changing
    their state permanently by their members.

FUNCTIONS
    Import the module and use Get-Help to get more details about the functions:

        Open-EnvironmentPanel
        Open-InventoryPanel
        Open-LogicalDiskPanel
        Open-ServicePanel
        Open-StartupCommandPanel

        Get-Uninstall
        Open-UninstallPanel
